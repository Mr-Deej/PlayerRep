# PlayerRep
Allow your players to give each other repuation on your server

## Default Commands
/rep - shows the player thier current reputation points

## Admin Commands
/rep show <player>
/rep add <player> {1-10}
/rep remove <player> {amount}
/rep reload

## Placeholders
%player% 
%repamount%
%reprec%
%reptaken%
%repadded%
